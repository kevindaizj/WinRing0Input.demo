<br/>

### 配置
1. Properties => Build => disable "prefer 32-bits" (use x64 dll)
2. Properties => Build Events => Post-build => 编译成功则复制Dll到bin目录。
3. 使用**管理员权限**运行
4. 输入法调整为美式英文键盘（待改进？）
